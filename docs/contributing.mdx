## Running locally

Firstly, fork or clone [https://github.com/DonnC/jawce](https://github.com/DonnC/jawce) repository.

```bash
git clone git@github.com:DonnC/jawce.git
```

Open the project with IntelliJ or any IDE of choice and install all the dependencies of the 3 packages in the order

1. session module
2. engine module
3. example project

<Warning>
    Before running, you must have your WhatsApp test credentials all set
</Warning>

> Checkout the getting started to [WhatsApp Cloud API](https://developers.facebook.com/docs/whatsapp/cloud-api/get-started/).

A good starting point is the [ChatBot](https://github.com/DonnC/jawce/tree/main/example/jchatbot) example chatbot, set your WhatsApp credentials in the `application.yml` properties file.

Run the project.
